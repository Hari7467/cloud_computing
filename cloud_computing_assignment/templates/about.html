About This Project

This cloud computing assignment demonstrates deploying a Flask application to Render.com. The project showcases fundamental concepts of cloud deployment, scalability, and web application development.

Project Objectives

  • Create an interactive web application using Flask
  • Implement responsive design principles
  • Deploy the application to a cloud platform (Render.com)
  • Demonstrate understanding of cloud computing concepts
